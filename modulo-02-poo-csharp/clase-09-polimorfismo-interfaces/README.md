# Clase 09 — Polimorfismo e Interfaces

## Objetivos

- Aplicar polimorfismo con clases abstractas e interfaces
- Definir e implementar interfaces
- Usar múltiples interfaces en una clase

## Conceptos clave

| Concepto | Descripción |
|----------|-------------|
| **Polimorfismo** | Múltiples comportamientos desde una misma referencia |
| **Clase abstracta** | Base que puede tener métodos concretos y abstractos |
| **Interfaz** | Contrato que las clases deben implementar |
| **Herencia múltiple** | Una clase puede implementar varias interfaces |

## Abstracta vs Interfaz

| Clase abstracta | Interfaz |
|-----------------|----------|
| Tiene constructores | No tiene constructores |
| Puede tener campos | No tiene campos |
| Herencia simple | Múltiples interfaces |
| Relación "es un" | Capacidad "puede hacer" |
