# Clase 07 — Encapsulamiento y Propiedades

## Objetivos

- Ocultar datos internos con modificadores de acceso
- Usar propiedades con getters y setters
- Validar datos en propiedades
- Diferenciar autoimplementadas, computadas y de solo lectura

## Conceptos clave

| Concepto | Descripción |
|----------|-------------|
| **Encapsulamiento** | Ocultar datos internos y controlar acceso |
| **`public` / `private`** | Modificadores de acceso |
| **Propiedad** | Miembro con get/set para acceder a datos |
| **Autoimplementada** | `{ get; set; }` — sin lógica extra |
| **Computada** | Solo get con expresión calculada |
| **`value`** | Parámetro implícito en el setter |
