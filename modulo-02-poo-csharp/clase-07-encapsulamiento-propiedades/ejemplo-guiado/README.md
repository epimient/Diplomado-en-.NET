# Ejemplo Guiado — Encapsulamiento con Empleados

## Pasos

```bash
dotnet new console -n Empleados
cd Empleados
```

Reemplazar `Program.cs` con el código de ejemplo, luego:

```bash
dotnet run
```

## Conceptos aplicados

- Campos privados con propiedades públicas
- Validación en setters (nombre, salario, edad)
- Propiedad computada `SalarioAnual`
- Propiedad de solo lectura `FechaIngreso`
- Manejo de excepciones con `try/catch`
