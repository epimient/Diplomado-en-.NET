# Ejercicio 1 — Básico: Clase Producto

## Enunciado

Crea una clase `Producto` con propiedades autoimplementadas para modelar un producto de tienda.

## Requisitos

- Propiedades autoimplementadas:
  - `public string Nombre { get; set; }`
  - `public double Precio { get; set; }`
  - `public int Stock { get; set; }`
- Constructor que reciba nombre, precio y stock
- Método `MostrarInfo()` que muestre todos los datos
- En `Main`, crear 2 productos y mostrar su info

## Salida esperada

```
Producto: Laptop - $1200.00 - Stock: 10
Producto: Mouse - $25.50 - Stock: 50
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Propiedades autoimplementadas correctas
- Constructor con parámetros
- Método `MostrarInfo()` funcional
