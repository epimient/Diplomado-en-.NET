# Clase 06 — Clases, Objetos y Constructores

## Objetivos

- Comprender POO y sus beneficios
- Definir clases con campos y métodos
- Crear objetos con constructores
- Diferenciar instancia vs estático

## Conceptos clave

| Concepto | Descripción |
|----------|-------------|
| **Clase** | Molde que define estructura y comportamiento |
| **Objeto** | Instancia concreta de una clase |
| **Constructor** | Método que inicializa el objeto al crearlo |
| **`new`** | Palabra clave para crear instancias |
| **`this`** | Referencia al objeto actual |
| **`static`** | Miembro que pertenece a la clase, no a la instancia |
