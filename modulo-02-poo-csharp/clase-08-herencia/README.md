# Clase 08 — Herencia

## Objetivos

- Crear jerarquías de clases con herencia
- Usar `base` para constructores y miembros
- Sobrescribir métodos con `virtual` / `override`
- Usar `sealed` y `protected`

## Conceptos clave

| Concepto | Descripción |
|----------|-------------|
| **Herencia** | Clase derivada hereda de clase base |
| **`base`** | Referencia al constructor o miembro base |
| **`virtual`** | Método que puede sobrescribirse |
| **`override`** | Sobrescritura de un método virtual |
| **`sealed`** | Evita herencia o sobrescritura |
| **`protected`** | Accesible desde clase base y derivadas |
