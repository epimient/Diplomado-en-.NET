# Ejercicio 3 — Reto: Calculadora de IMC

## Enunciado

Crea un programa que calcule el Índice de Masa Corporal (IMC) y muestre la categoría correspondiente.

## Requisitos

- Fórmula: IMC = peso / (altura × altura)
- Peso en kg (double), altura en metros (double)
- Mostrar IMC con 1 decimal
- Mostrar categoría según tabla

## Tabla de categorías

| IMC | Categoría |
|-----|-----------|
| < 18.5 | Bajo peso |
| 18.5 - 24.9 | Peso normal |
| 25.0 - 29.9 | Sobrepeso |
| ≥ 30.0 | Obesidad |

## Entrada esperada

```
Ingresa tu peso (kg): 70
Ingresa tu altura (m): 1.75
```

## Salida esperada

```
Tu IMC es: 22.9
Categoría: Peso normal
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Fórmula del IMC correcta
- Categorización correcta
- Resultados formateados adecuadamente
