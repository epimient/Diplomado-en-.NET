# Ejercicio 1 — Básico: Operaciones aritméticas

## Enunciado

Crea un programa que pida dos números enteros y muestre la suma, resta, multiplicación, división y residuo.

## Requisitos

- Usar `int` para los números
- Usar `Console.ReadLine()` y `int.Parse()`
- Mostrar los 5 resultados

## Entrada esperada

```
Ingresa el primer número: 10
Ingresa el segundo número: 3
```

## Salida esperada

```
Suma: 13
Resta: 7
Multiplicación: 30
División: 3
Residuo: 1
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Captura ambos números correctamente
- Muestra los 5 resultados correctos
