# Ejercicio 2 — Intermedio: Área y perímetro del rectángulo

## Enunciado

Crea un programa que calcule el área y el perímetro de un rectángulo usando valores decimales.

## Requisitos

- Usar `double` para base y altura
- Área = base × altura
- Perímetro = 2 × (base + altura)
- Mostrar resultado con 2 decimales

## Entrada esperada

```
Ingresa la base: 5.5
Ingresa la altura: 3.2
```

## Salida esperada

```
Área: 17.60
Perímetro: 17.40
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Usa tipo `double`
- Resultados con 2 decimales
- Fórmulas correctas
