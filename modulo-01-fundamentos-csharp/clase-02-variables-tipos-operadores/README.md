# Clase 02 — Variables, Tipos de Datos y Operadores

## Objetivos

- Declarar y usar variables con diferentes tipos
- Aplicar operadores aritméticos, comparación y lógicos
- Convertir entre tipos de datos

## Tipos principales

| Tipo | Ejemplo |
|------|---------|
| `int` | `int edad = 25;` |
| `double` | `double altura = 1.75;` |
| `string` | `string nombre = "Ana";` |
| `bool` | `bool activo = true;` |
