# Ejemplo Guiado — Calculadora de Promedio

## Pasos

```bash
dotnet new console -n CalculadoraPromedio
cd CalculadoraPromedio
```

Reemplazar `Program.cs` con el código de ejemplo, luego:

```bash
dotnet run
```

## Conceptos aplicados

- Declaración de variables (`string`, `double`)
- Conversión con `Convert.ToDouble()`
- Operadores aritméticos: suma, división
- Operador ternario para estado
- Interpolación de cadenas con formato `:F2`
