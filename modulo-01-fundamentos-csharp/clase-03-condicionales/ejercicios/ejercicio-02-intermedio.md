# Ejercicio 2 — Intermedio: Calculadora de descuento

## Enunciado

Crea un programa que calcule el precio final de un producto aplicando un descuento según su precio original.

## Requisitos

- Usar `if / else if / else`
- Mostrar el descuento aplicado y el precio final

## Tabla de descuentos

| Precio original | Descuento |
|-----------------|-----------|
| < $50 | 0% |
| $50 - $100 | 10% |
| $100 - $500 | 20% |
| > $500 | 30% |

## Entrada esperada

```
Ingresa el precio del producto: 250
```

## Salida esperada

```
Precio original: $250.00
Descuento aplicado: 20%
Precio final: $200.00
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Aplica el descuento correcto según el rango
- Muestra el porcentaje y el precio final
- Resultado formateado con 2 decimales
