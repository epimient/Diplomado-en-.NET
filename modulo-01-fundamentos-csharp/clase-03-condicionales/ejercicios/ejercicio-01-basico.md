# Ejercicio 1 — Básico: Par o impar

## Enunciado

Crea un programa que pida un número entero y determine si es par o impar.

## Requisitos

- Usar el operador `%` (módulo)
- Usar `if/else`

## Entrada esperada

```
Ingresa un número: 7
```

## Salida esperada

```
El número 7 es impar.
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Detecta correctamente pares e impares
- Muestra el mensaje con el número ingresado

---

*Pista: un número es par si `numero % 2 == 0`.*
