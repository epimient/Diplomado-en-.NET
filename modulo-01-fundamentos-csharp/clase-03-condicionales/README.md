# Clase 03 — Condicionales

## Objetivos

- Decidir el flujo del programa con `if/else` y `switch`
- Usar el operador ternario para asignaciones simples
- Validar datos con condicionales

## Estructuras vistas

| Estructura | Uso |
|------------|-----|
| `if / else if / else` | Condiciones variadas |
| `? :` | Asignación condicional simple |
| `switch` | Múltiples valores fijos |
