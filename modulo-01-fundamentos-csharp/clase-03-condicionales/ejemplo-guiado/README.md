# Ejemplo Guiado — Calculadora con Switch

## Pasos

```bash
dotnet new console -n CalculadoraSwitch
cd CalculadoraSwitch
```

Reemplazar `Program.cs` con el código de ejemplo, luego:

```bash
dotnet run
```

## Conceptos aplicados

- `switch` para menú de operaciones
- `if` anidado para validar división entre cero
- `Convert.ToDouble` y `Convert.ToInt32`
- `return` para salir del programa en caso de error
