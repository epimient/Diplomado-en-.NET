# Clase 01 — Introducción a .NET

## Objetivos

- Entender qué es .NET y su ecosistema
- Instalar y verificar el SDK
- Usar la CLI de .NET
- Crear y ejecutar un proyecto de consola
- Escribir programas con entrada y salida

## Comandos clave

```bash
dotnet --version
dotnet new console -n MiApp
dotnet run
```

## Métodos de Console

- `Console.WriteLine()` — Imprime con salto de línea
- `Console.Write()` — Imprime sin salto de línea
- `Console.ReadLine()` — Lee entrada del usuario
- Interpolación: `$"Hola, {nombre}"`
