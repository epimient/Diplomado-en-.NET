# Ejercicio 2 — Intermedio: Perfil del usuario

## Enunciado

Crea un programa que solicite nombre, edad y ciudad del usuario, luego muestre una presentación completa.

## Requisitos

- Pedir al menos 3 datos diferentes
- Mostrar una presentación usando todos los datos

## Entrada esperada

```
Nombre: Carlos
Edad: 25
Ciudad: Medellín
```

## Salida esperada

```
Hola, Carlos! Tienes 25 años y vives en Medellín.
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Captura correctamente los 3 datos
- La salida incluye todos los datos ingresados
- Usa interpolación de cadenas
