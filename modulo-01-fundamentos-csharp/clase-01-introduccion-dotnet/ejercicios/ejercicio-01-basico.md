# Ejercicio 1 — Básico: Saludo personal

## Enunciado

Crea un programa que pida el nombre del usuario y muestre un saludo personalizado.

## Requisitos

- Usar `Console.Write` o `Console.WriteLine`
- Usar `Console.ReadLine`
- Usar interpolación de cadenas `$`

## Entrada esperada

```
¿Cómo te llamas? Ana
```

## Salida esperada

```
Hola, Ana! Bienvenida a .NET.
```

## Criterios de evaluación

- El programa compila y ejecuta sin errores
- Lee correctamente el nombre ingresado
- Muestra el saludo con el nombre ingresado
