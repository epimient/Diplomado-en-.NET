# Ejercicio 3 — Reto: Menú de presentación

## Enunciado

Crea un programa que muestre un menú con 3 opciones. El usuario elige una opción y el programa ejecuta la acción correspondiente.

## Requisitos

- Mostrar un menú con opciones numeradas
- Usar `if` o `switch` para evaluar la opción
- El programa debe ejecutarse sin errores

## Menú esperado

```
===== MENÚ PRINCIPAL =====
1. Saludar
2. Mostrar fecha y hora
3. Salir
==========================
Elige una opción:
```

## Comportamiento esperado

- Opción 1: Pide nombre y saluda
- Opción 2: Muestra la fecha y hora actual usando `DateTime.Now`
- Opción 3: Muestra "¡Hasta luego!" y termina

## Criterios de evaluación

- Compila y ejecuta sin errores
- El menú se muestra correctamente
- Cada opción ejecuta la acción correcta
- El programa maneja opciones inválidas (muestra "Opción no válida")
