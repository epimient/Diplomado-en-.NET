# Ejemplo Guiado — Sistema de Notas con Métodos

## Pasos

```bash
dotnet new console -n SistemaNotas
cd SistemaNotas
```

Reemplazar `Program.cs` con el código de ejemplo, luego:

```bash
dotnet run
```

## Conceptos aplicados

- Métodos con y sin retorno
- Parámetros: `string`, `int`, `double`, `double[]`
- Ciclo `for` para capturar notas
- Ciclo `foreach` para sumar notas
- Condicionales para determinar estado
- Modularización: cada tarea en su propio método
