# Ejercicio 1 — Básico: Calculadora con métodos

## Enunciado

Crea un programa con un método para cada operación: `Sumar`, `Restar`, `Multiplicar`, `Dividir`. Cada método recibe dos números y retorna el resultado.

## Requisitos

- Cada operación debe ser un método separado
- El menú principal usa `switch`
- Usar `do while` para mantener el menú activo

## Entrada esperada

```
1. Sumar
2. Restar
3. Multiplicar
4. Dividir
5. Salir
Elige: 1
Ingresa número 1: 10
Ingresa número 2: 5
Resultado: 15
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Cada operación es un método con retorno `double`
- Menú funcional con ciclo
- Maneja división entre cero
