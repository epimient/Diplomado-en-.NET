# Ejemplo Guiado — Menú interactivo con while

## Pasos

```bash
dotnet new console -n MenuInteractivo
cd MenuInteractivo
```

Reemplazar `Program.cs` con el código de ejemplo, luego:

```bash
dotnet run
```

## Conceptos aplicados

- `while` para mantener el menú activo
- `switch` dentro del ciclo para manejar opciones
- `for` para contar y generar tablas de multiplicar
- Validación de opción con `default`
