# Ejercicio 3 — Reto: Dibujar figuras con asteriscos

## Enunciado

Crea un programa que muestre un menú para dibujar figuras geométricas usando asteriscos con ciclos anidados.

## Opciones del menú

```
1. Cuadrado (n × n)
2. Triángulo rectángulo
3. Triángulo invertido
4. Salir
```

## Requisitos

- Usar ciclos anidados (`for` dentro de `for`)
- El usuario elige la figura y el tamaño

## Ejemplos de salida

### Cuadrado (n = 4)

```
****
****
****
****
```

### Triángulo rectángulo (n = 4)

```
*
**
***
****
```

### Triángulo invertido (n = 4)

```
****
***
**
*
```

## Criterios de evaluación

- Compila y ejecuta sin errores
- Usa ciclos anidados
- Figuras dibujadas correctamente
- Menú funcional con opciones
