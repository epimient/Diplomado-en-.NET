# Clase 04 — Ciclos (Bucles)

## Objetivos

- Repetir código con `for`, `while`, `do while`, `foreach`
- Controlar flujo con `break` y `continue`
- Elegir el ciclo adecuado

## Estructuras vistas

| Ciclo | Cuándo usarlo |
|-------|---------------|
| `for` | Iteraciones conocidas |
| `while` | Condición al inicio |
| `do while` | Condición al final (mínimo 1 vez) |
| `foreach` | Recorrer colecciones |
