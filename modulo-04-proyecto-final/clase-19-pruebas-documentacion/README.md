# Clase 19 — Pruebas y Documentación

## Descripción

Prueba la API del proyecto final desde Swagger, Thunder Client y curl. Prepara la documentación, capturas de pantalla, video demo y README.md profesional.

## Qué aprenderás

- Probar endpoints desde Swagger UI
- Probar con Thunder Client o Postman
- Probar con curl desde terminal
- Capturar evidencia de pruebas
- Escribir README.md profesional
- Grabar video de demostración

## Requisitos

- API funcional (clase 18)
- Thunder Client, Postman o curl
- Herramienta de captura de pantalla
- OBS Studio o grabador de pantalla

## Instrucciones

```bash
cd proyecto-final/MiProyectoFinal
dotnet run
```

Probar cada endpoint y capturar resultados.
