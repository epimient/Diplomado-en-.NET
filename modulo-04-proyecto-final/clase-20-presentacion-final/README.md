# Clase 20 — Presentación Final

## Descripción

Última clase del diplomado. Los equipos presentan su proyecto final, demuestran la API en vivo y comparten aprendizajes. Es el cierre del curso.

## Qué aprenderás

- Preparar una presentación técnica efectiva
- Demostrar una API en vivo sin errores
- Responder preguntas técnicas del jurado
- Recibir y dar retroalimentación constructiva
- Reflexionar sobre el proceso de aprendizaje

## Requisitos

- Proyecto final completo y funcional
- Presentación en PowerPoint, Google Slides o Canva
- Video de demostración grabado
- Repositorio de GitHub actualizado
- Capturas de prueba en `screenshots/`

## Instrucciones

```bash
cd proyecto-final/MiProyectoFinal
dotnet run
```

Tener Swagger UI listo en `http://localhost:5000/swagger`.
