# Diplomado Programación con .NET — Ejercicios Formativos

**Repositorio de ejercicios evaluados** del Diplomado Programación con .NET.

## Estructura

```
diplomado-ejercicios/
├── README.md
├── ejercicio-01-luhn/      → Entrega: domingo semana 1
├── ejercicio-02-poo/       → Entrega: domingo semana 2
└── ejercicio-03-clima-api/ → Entrega: domingo semana 3
```

## Evaluación

Cada ejercicio vale **16.66%** de la nota final. El promedio de los 3 ejercicios = **50%** de la nota total (formativa).

## Entrega

1. Crear carpeta con su nombre dentro del ejercicio
2. Subir todo el proyecto `.csproj` + `Program.cs` + archivos adicionales
3. Incluir captura de pantalla de la terminal o Swagger mostrando funcionamiento
4. Fecha límite: domingo 11:59 PM de la semana correspondiente

## Política

- Ejercicios entregados después del domingo → penalización del 50%
- Ejercicios que no compilan con `dotnet build` → nota máxima 3.0
- Ejercicios copiados entre compañeros → 0 automático
